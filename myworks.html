My Interactive Applications

Welcome to my works page! Here you'll find two JavaScript applications I built for my portfolio. One runs entirely in the browser, and the other fetches live data from a public API.

App 1
✅

To-Do List Manager

📝

Type: Pure JavaScript App (No External APIs)

Description: A fully functional to-do list application that allows you to add, complete, and delete tasks. Your tasks are saved locally in your browser so they persist even after you close the page!

Features:

  • Add new tasks with a title
  • Mark tasks as complete/incomplete
  • Delete individual tasks
  • Clear all completed tasks at once
  • Tasks saved automatically with localStorage
  • Task counter showing remaining tasks

Country Explorer

Type: API-Powered App (REST Countries API)

Description: Explore countries from around the world! Search for any country to see detailed information including capital, population, region, and flag.
